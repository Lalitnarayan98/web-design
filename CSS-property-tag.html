CSS property related html attribute

  1. ID
    Identify a unique element/tag in html
    ex.
      id="id1"
  2. CLASS
    Identify a group of element/tag in html
    ex.
      class="class1"
  3. DIV
    Used for select a division or make a division
    ex.
      <div > </div >
  4. Section
    Used for select a section or make a division
    ex.
      <section> </section>
